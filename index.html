Kaltura Avatar SDK

Embed AI avatar conversations in any website. One script tag, a few lines of code, and you have a talking AI avatar with real-time speech, dynamic context, and full event control.

Use with AI Coding Agents

This repo is designed to work as an instant skill for AI coding agents. Point your agent at this repo and tell it what to build — it understands the full SDK, all advanced patterns, and how to write the avatar's Knowledge Base prompt.

Claude Code / Claude Projects: Add this repo as context, then prompt:

"Build me a customer onboarding avatar using this SDK. My client ID is 123456, flow ID is agent-7."

ChatGPT / Copilot / Any Agent: Paste AGENTS.md into your system prompt, then ask away.

The agent generates both the app code (HTML + JS with DPP injection and spoken command detection) and the Knowledge Base prompt (paste into Kaltura Studio).

Quick Start

<div id="avatar" style="width:800px;height:600px"></div>
<script src="https://zoharbabin.github.io/kaltura-avatar-sdk/sdk/kaltura-avatar-sdk.js"></script>
<script>
const sdk = new KalturaAvatarSDK({
  clientId: 'YOUR_CLIENT_ID',
  flowId: 'YOUR_FLOW_ID',
  container: '#avatar'
});

sdk.on('showing-agent', () => {
  setTimeout(() => sdk.injectPrompt(JSON.stringify({ greeting: "Hello!" })), 500);
});

sdk.on('agent-talked', (data) => {
  console.log('Avatar said:', data.agentContent || data);
});

await sdk.start();
</script>

Key Capabilities

All of these are documented in AGENTS.md with working code examples:

  • Dynamic Prompt Injection (DPP) — inject JSON context at runtime to control avatar behavior per session
  • Avatar Spoken Commands — avatar speaks trigger phrases that fire JavaScript functions
  • Multi-persona — one Knowledge Base, multiple avatars selected via DPP
  • Visual effects — show images, popup links, collect emails (configured in Knowledge Base)
  • Transcripts — built-in recording with export to text, markdown, or JSON
  • RICECO framework — structured guide for writing great Knowledge Base prompts
