Embed AI Avatar Conversations in Any Website

A lightweight JavaScript SDK to add interactive AI avatars to your app. Real-time speech, dynamic context injection, and full event system.

See Live Demos View on GitHub 
// 3 lines to get a talking avatar on your page
const sdk = new KalturaAvatarSDK({
  clientId: 'YOUR_CLIENT_ID',
  flowId: 'YOUR_FLOW_ID',
  container: '#avatar'
});

await sdk.start();

// Listen to what the avatar says
sdk.on('agent-talked', (data) => {
  console.log('Avatar:', data.agentContent);
});

// Inject dynamic context at runtime
sdk.injectPrompt(JSON.stringify({ role: 'interviewer', topic: 'JavaScript' }));

Get Started in 5 Minutes

1

Include the SDK

Add one script tag. No build step required.

<script src="kaltura-avatar-sdk.min.js">
2

Create & Start

Pass your credentials and a container element.

new KalturaAvatarSDK({...}).start()
3

Customize with DPP

Inject JSON context to shape avatar behavior.

sdk.injectPrompt(json)
